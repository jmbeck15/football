Compilation
-----------
Under Windows, the program compiles just fine with Qt Creator 3.0.0 and MSVC 2012 with Qt 5.2.0.

Under Linux, the program spews tons of vtable errors with Qt Creator 3.0 and gcc 4.8.1 with Qt 5.2.0.
